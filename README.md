# PHP Work Wechat
PHP 开发的企业微信消息加密和解密SDK，简化企业微信API开发步骤。

## 说明文档

### 安装 
```bash
composer require "douyuxingchen/php-work-wechat"
```

### 更新
```bash
composer update "douyuxingchen/php-work-wechat" --ignore-platform-reqs
```

## 特性
- 消息：为消息提供加密和解密服务
